  • IE6 中，第一个浮动到父元素边上的元素，如果含有该方向的 margin 值，那么 margin 会以双倍显示。即：浮动元素的左边距在 IE6 上为所设定的左边距的两倍。这个问题只会发生在浮动行的第一个浮动元素上。准确的说：应该是每一行的第一个元素都会受此影响。
    为了解决该问题，需要给浮动元素添加属性 display: inline，即可解决。
